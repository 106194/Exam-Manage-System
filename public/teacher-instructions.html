Create Exam

You can create an exam by choosing level (SSC/HSC/Others), subject, chapter, duration, and set whether it’s public or password-protected. For the running exams, you can't see the update. After the exams have finished, you can see the leaderboard. Be careful about selecting time (this must be consistent). Set at least one question for the exam; otherwise, the exam will be considered invalid and discarded.

Set Questions

After creating an exam, you will be redirected to the Set Question page. There, add each question and its options. Only one of text or image can be used per question/option. You must select question or option type (image) to upload image files. Be careful about uploading files. They must be images — preferably JPG or PNG.

Question Suggestion

Use previous exams or your own stored content as a reference. Ensure clarity, relevance, and fairness in question marks. Be careful when adding questions from previous sources — they will be added with the same points and options which were set in the past by you or others.

Update Settings

You can edit your Full Name, Institution, Bio, and Profile Picture. You can reset your password using email verification if forgotten. You can't change your username directly — contact the admin to request a username change.

Privacy & Policy: Uploaded questions and exam settings are private and stored securely. Don’t share protected exam credentials with unauthorized users. Don't share your password or email with others for security reasons.
